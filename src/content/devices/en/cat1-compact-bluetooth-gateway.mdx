<a href="https://www.lansitec.com/products/cat-1-compact-bluetooth-gateway/" target="_blank" rel="noopener noreferrer">Cat-1 Compact Bluetooth Gateway</a> receives nearby Bluetooth beacons, sensors, or controllers&#39; data, restructure, and forward it to a server via Cat-1.<br />
It supports iBeacon, Eddystone, private protocols, and the scan and response feature.<br />
The integrated 600mAh lithium-ion rechargeable battery provides an operational duration of approximately 10 hours of operation when no external power source is available.<br />

## Prerequisites

To continue with this guide we will need the following:

- <a href="https://www.lansitec.com/products/cat-1-badge-tracker/" target="_blank">Cat-1 Compact Bluetooth Gateway</a>
- <a href="https://www.lansitec.com/products/bluetooth-beacon/" target="_blank" rel="noopener noreferrer">Bluetooth Beacon</a>

- <a href="https://localhost" target="_blank" rel="noopener noreferrer">ThingsBoard account</a>

## Configuration

You will need to have access to ThingsBoard Professional Edition. The easiest way is to use <a href="https://thingsboard.io/installations/choose-region/" target="_blank" rel="noopener noreferrer">ThingsBoard Cloud</a> server.
The alternative option is to install ThingsBoard using <a href="/docs/user-guide/install/pe/installation-options/" target="_blank" rel="noopener noreferrer">installation guide</a>.

## Create device on ThingsBoard

<Steps>

1. Go to the **Devices** page and click on **Add device** button.
2. Fill in the device information and click the **Next: Credentials** button.
3. When adding a device to ThingsBoard, fill in the IMEI number of the device (e.g., 862990061908629) as the access token. This device is directly connected to the ThingsBoard platform through the MQTT protocol and pushes data in JSON format.
4. Fill in the device **Access token** and click the **Add** button.
5. Click on the device to view detailed information.
6. Click on **Latest telemetry** to view device reported data.

</Steps>

## Check data on ThingsBoard

So, the device was added and if it sends any data - it should appear in the <a href="/docs/user-guide/ui/devices/" target="_blank" rel="noopener noreferrer">devices</a>.
To check it you may open **Devices** page in **Entities** section. The device should be in devices list. You can check the data by click on it and open tab **Attributes** or **Latest telemetry**.

<ImageGallery images={[{"src":"/src/assets/devices-library/ready-to-go-devices/lansitec-lorawan/cat1-beacon1.png","alt":"You should now see data from the device.","caption":"You should now see data from the device."}]} />

In order to get more user-friendly view - you can use <a href="/docs/user-guide/dashboards/" target="_blank" rel="noopener noreferrer">dashboards</a>.
You can download a simple dashboard for this device, it is configured to display a data from "latitude" and "longitude" telemetry keys for device with name "*Devices*".

ThingsBoard provides the ability to create and customize interactive visualizations (dashboards) for monitoring and managing data and devices.
Through ThingsBoard dashboards, you can efficiently manage and monitor your IoT devices and data. So, we will create the dashboard, for our device.

To add the dashboard to ThingsBoard, we need to import it. To import a dashboard, follow these steps:

- First download the <a href="/docs/devices-library/resources/dashboards/lansitec-lorawan/cat1_check_and_control_device_data_dashboard.json" target="_blank" rel="noopener noreferrer">Check and control device data dashboard</a> file.

<Steps>

1. Navigate to the "**Dashboards**" page. By default, you navigate to the dashboard group "All". Click on the "**plus**" icon in the top right corner. Select "**Import dashboard**".
2. In the dashboard import window, upload the JSON file and click "**Import**" button.
3. Dashboard has been imported.

</Steps>

To do this, follow these steps:

<Steps>

1. Open the dashboard and enter edit mode. Click the "**Entity aliases**" icon, then in the pop-up window click the "**Edit alias**" icon next to the alias.
2. In edit alias window select your device from dropdown list and save entity alias.
3. Apply all changes.

</Steps>

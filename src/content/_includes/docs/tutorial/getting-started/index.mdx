<Lede> This guide introduces the core ThingsBoard concepts and helps you get started with IoT data in just a few minutes. 🚀 </Lede>

By completing it, you will learn how to:
- Connect a device to ThingsBoard
- Publish telemetry data
- Visualize data on interactive dashboards
- Configure alarms and notifications
- Add a customer and securely share data with them

<Aside type="tip">
At the end of each page, you will find an interactive checkbox to track your progress in the training materials, along with navigation buttons.
Using the tracker is optional, but it can help you remember where you left off and mark topics for later review.

All data is stored locally in your browser and is not sent to or stored by ThingsBoard.
</Aside>

Alarm widgets in ThingsBoard are a special group of dashboard widgets designed to display, monitor, and manage alarms generated by entities such as devices, assets, or rule chains.
They provide visual and interactive tools to help users track abnormal conditions, system faults, or important events in real time.

These widgets are typically used in dashboards for operations, maintenance, or monitoring teams — allowing them to quickly see what went wrong, when, and where.

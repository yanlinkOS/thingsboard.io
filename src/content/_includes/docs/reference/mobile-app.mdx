The ThingsBoard Mobile Application is an open-source project based on [Flutter](https://flutter.dev/) that lets you build IoT mobile applications with minimum coding effort.
It is available for both iOS and Android and licensed under **Apache 2.0**.

## Features

- Browse and monitor IoT dashboards on your phone or tablet
- Manage devices and view alarms
- Customize the app with your branding (white-labeling)
- OAuth 2.0 and self-registration support
- QR code scanner for device provisioning
- Localization support

## Source code

Both Community and Professional Edition apps are open-source:

| Edition | GitHub repository |
|---|---|
| Community Edition | [flutter_thingsboard_app](https://github.com/thingsboard/flutter_thingsboard_app) |
| Professional Edition | [flutter_thingsboard_pe_app](https://github.com/thingsboard/flutter_thingsboard_pe_app) |

Install and run ThingsBoard IoT Gateway using Docker Compose on Windows.

## Prerequisites

- [Docker Desktop for Windows](https://docs.docker.com/desktop/setup/install/windows-install/) is installed.
- A running ThingsBoard instance. Use [ThingsBoard Cloud](https://thingsboard.cloud/signup) or install ThingsBoard locally — see the <DocLink product="CE" path="installation/" bold={false}>installation guide</DocLink>.

## Installation

<Steps>
  <ol>
    <li>
      **Create a Gateway device and download the `docker-compose.yml` file.**

      In ThingsBoard, go to **Entities** > **Gateways** and click the **+** icon in the upper-right corner.

      In the **Launch command** window that opens, click **Download** to save the `docker-compose.yml` file.

## Running in the background

To run the Gateway in detached (background) mode:

```bash
docker compose up -d
```

To view the Gateway logs after detaching:

```bash
docker logs -f tb-gateway
```

## Upgrading

To update the Gateway to the latest image:

```bash
docker compose down
docker pull thingsboard/tb-gateway
docker compose up -d
```

For a full upgrade guide, see <DocLink product={props.product} path="installation/upgrade-instructions/?upgradeMethod=docker" bold={false}>Upgrade instructions</DocLink>.

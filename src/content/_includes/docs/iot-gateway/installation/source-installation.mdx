Build and run ThingsBoard IoT Gateway directly from source on Ubuntu or Debian-based systems.
This method is recommended for development and local customization.

## Prerequisites

- Python 3.10 or higher is installed.
- `git` is available on the system.

## Installation from source

<Steps>
  <ol>
    <li>
      **Install system dependencies.**

      ```bash
      sudo apt install python3-dev python3-pip python3-venv libglib2.0-dev git
      ```
    </li>
    <li>
      **Clone the repository.**

      ```bash
      git clone --recurse-submodules https://github.com/thingsboard/thingsboard-gateway.git --depth 1
      ```
    </li>
    <li>
      **Enter the project directory.**

      ```bash
      cd thingsboard-gateway
      ```
    </li>
    <li>
      **Create and activate a virtual environment.**

      ```bash
      python3 -m venv venv && source venv/bin/activate
      ```
    </li>
    <li>
      **Install Python dependencies.**

      ```bash
      pip install -r requirements.txt
      ```
    </li>
    <li>
      **Install the Gateway package.**

      ```bash
      python setup.py install
      ```
    </li>
    <li>
      **Create the logs directory.**

      ```bash
      mkdir logs
      ```
    </li>
    <li>
      **Configure the Gateway.** Open the configuration file and set your ThingsBoard host and access token:

      ```bash
      thingsboard-gateway/thingsboard_gateway/config/tb_gateway.json
      ```

      For a full description of all configuration options, see the <DocLink product={props.product} path="config/general" bold={false}>configuration guide</DocLink>.
    </li>
    <li>
      **Start the Gateway to verify the installation.**

      ```bash
      python3 ./thingsboard_gateway/tb_gateway.py
      ```
    </li>
  </ol>
</Steps>

## Build a local Docker image

<Steps>
  <ol>
    <li>
      **Copy the Dockerfile to the project root.**

      ```bash
      cp docker/Dockerfile .
      ```
    </li>
    <li>
      **Build the image.**

      ```bash
      docker build -t local-gateway .
      ```
    </li>
  </ol>
</Steps>

## Hot Reloader

When developing custom extensions, enable Hot Reloader to restart the Gateway automatically
whenever a project file changes:

```bash
python3 ./thingsboard_gateway/tb_gateway.py true
```

<Aside type="note">
  Hot Reloader is intended for development only. Do not use it in production deployments.
</Aside>

## Connect

Use any MQTT client. The examples below use <a href="https://mosquitto.org/download/" target="_blank" rel="noopener noreferrer">mosquitto</a>, a widely used open-source CLI client.

<Tabs>
  <TabItem label="Subscribe">
    ```bash
    mosquitto_sub -d -h demo.tbmq.io -p 1883 -q 1 \
      -t sensors/temperature -u demo
    ```
  </TabItem>
  <TabItem label="Publish">
    ```bash
    mosquitto_pub -d -h demo.tbmq.io -p 1883 -q 1 \
      -t sensors/temperature -m 25 -u demo
    ```
  </TabItem>
  <TabItem label="Subscribe over TLS">
    ```bash
    mosquitto_sub -d -h demo.tbmq.io -p 8883 -q 1 \
      -t sensors/temperature -u demo \
      --cafile tbmq-demo-root-ca.pem
    ```
  </TabItem>
</Tabs>

## Add image

Images can be added in [image file format](#upload-image) (PNG, JPEG, GIF, etc.) or [JSON file format](#import-image-from-json).

### Upload image

To upload an image in image file format:

<Steps>
1. Go to the **White labeling** page and click the **Image gallery** tab. Then click **Upload image** in the top right corner.

   <ImageGallery product={props.product} images={[
     { src: '/src/assets/images/docs/mqtt-broker/white-labeling/upload-image-1.png', alt: 'Open White labeling Image gallery tab and click Upload image' },
   ]} />

2. Select an image or drag it into the **Upload image** window, then click **Upload**.

   <ImageGallery product={props.product} images={[
     { src: '/src/assets/images/docs/mqtt-broker/white-labeling/upload-image-2.png', alt: 'Select or drag an image file and click Upload' },
   ]} />

3. The image has been uploaded.

   <ImageGallery product={props.product} images={[
     { src: '/src/assets/images/docs/mqtt-broker/white-labeling/upload-image-3.png', alt: 'Image uploaded successfully' },
   ]} />
</Steps>

## Image operations

Each image supports the following operations via the icons next to the image name: [download](#download-image), [export to JSON](#export-image-to-json), [edit](#edit-image), and [delete](#delete-image).

### Download image

Download an image in image file format — the method depends on the current view mode:

- **List view** — click the **Download image** icon next to the image name.
- **Grid view** — hover over the image and click the **Download image** icon.

The image file will be saved to your computer.

<ImageGallery product={props.product} images={[
  { src: '/src/assets/images/docs/mqtt-broker/white-labeling/download-image-1.png', alt: 'Download image from list view', caption: 'List view' },
  { src: '/src/assets/images/docs/mqtt-broker/white-labeling/download-image-2.png', alt: 'Download image from grid view', caption: 'Grid view' },
]} />

### Export image to JSON

Export an image to JSON — the method depends on the current view mode:

- **List view** — click the **Export image to JSON** icon next to the image name.
- **Grid view** — hover over the image and click the **Export image to JSON** icon.

The JSON file will be saved to your computer.

<ImageGallery product={props.product} images={[
  { src: '/src/assets/images/docs/mqtt-broker/white-labeling/export-image-1.png', alt: 'Export image to JSON from list view', caption: 'List view' },
  { src: '/src/assets/images/docs/mqtt-broker/white-labeling/export-image-2.png', alt: 'Export image to JSON from grid view', caption: 'Grid view' },
]} />

### Edit image

To open the editing window, click the **Edit image** icon next to the image name (in grid view, hover over the image first).
From this window you can rename the image, download it, export it to JSON, or [update it](#update-image).

To rename an image, enter a new name in the editing window and click the **Save** icon.

<ImageGallery product={props.product} images={[
  { src: '/src/assets/images/docs/mqtt-broker/white-labeling/edit-image-1.png', alt: 'Click Edit image icon to open the editing window', caption: 'Open Edit image window' },

  { src: '/src/assets/images/docs/mqtt-broker/white-labeling/edit-image-3.png', alt: 'Image name updated', caption: 'Name changed' },
]} />

#### Update image

Updating an image is useful when one image serves as the background or logo in multiple places — updating it once propagates the change everywhere automatically.

To update an image, click **Update image** in the editing window. Select a new image or drag it into the window and click **Update**.

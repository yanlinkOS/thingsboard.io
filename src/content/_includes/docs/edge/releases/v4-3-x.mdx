## v4.3.0 (Jan 21, 2026)

**Major** release with everything from [TB CE v4.3.0](/docs/releases/releases-table/v4-3-x/):

**Improvements**

* <a href="https://github.com/thingsboard/thingsboard/pull/13998" target="_blank" rel="noopener noreferrer">#13998</a> **AI Model synchronization:** Added automatic synchronization of AI models to Edge instances for local AI processing capabilities by <a href="https://github.com/jekka001" target="_blank" rel="noopener noreferrer">@jekka001</a>;
* <a href="https://github.com/thingsboard/thingsboard/pull/14352" target="_blank" rel="noopener noreferrer">#14352</a> **User and credential synchronization for Edge:** Added support for syncing User entities and UserCredentials from Edge to Cloud, enabling centralized user management by <a href="https://github.com/volodymyr-babak" target="_blank" rel="noopener noreferrer">@volodymyr-babak</a>;
* <a href="https://github.com/thingsboard/thingsboard/pull/14447" target="_blank" rel="noopener noreferrer">#14447</a> **Entity deletion sync for Edge:** Modified edge-to-cloud synchronization to propagate actual entity deletions (assets, devices, dashboards, entity views) instead of just unassigning them by <a href="https://github.com/MazurenkoNick" target="_blank" rel="noopener noreferrer">@MazurenkoNick</a>;
* <a href="https://github.com/thingsboard/thingsboard/pull/14603" target="_blank" rel="noopener noreferrer">#14603</a> **Edge Events optimization:** Added merge and filter functionality for duplicate Edge Events in the downlink message pipeline, reducing the number of events sent to edge devices by consolidating attribute updates and removing exact duplicates by <a href="https://github.com/volodymyr-babak" target="_blank" rel="noopener noreferrer">@volodymyr-babak</a>;
* <a href="https://github.com/thingsboard/thingsboard-edge/pull/209" target="_blank" rel="noopener noreferrer">#209</a> **Downlink duplicate merging:** Implemented merge and filter functionality for duplicate downlink messages, optimizing Edge communication by <a href="https://github.com/volodymyr-babak" target="_blank" rel="noopener noreferrer">@volodymyr-babak</a>;
* <a href="https://github.com/thingsboard/thingsboard-edge/pull/208" target="_blank" rel="noopener noreferrer">#208</a> **Refactor BaseCloudManagerService and its implementations** by <a href="https://github.com/MazurenkoNick" target="_blank" rel="noopener noreferrer">@MazurenkoNick</a>;
* <a href="https://github.com/thingsboard/thingsboard/pull/14489" target="_blank" rel="noopener noreferrer">#14489</a> **Edge update deduplication:** Reduced duplicate entity updates in edge processors by introducing generateUniqueNameIfDuplicateExists functionality by <a href="https://github.com/MazurenkoNick" target="_blank" rel="noopener noreferrer">@MazurenkoNick</a>;

**Bug fixes**

* <a href="https://github.com/thingsboard/thingsboard/pull/13843" target="_blank" rel="noopener noreferrer">#13843</a> **Improved Edge session cleanup** to prevent resource leaks and message backlog due to unstable network conditions and Kafka busy timeout by <a href="https://github.com/bcblr1993" target="_blank" rel="noopener noreferrer">@bcblr1993</a>.

## Create a webhook workflow in Microsoft Teams

<Steps>
1. Sign in to [Microsoft Teams](https://www.microsoft.com/en-us/microsoft-teams/group-chat-software/) and navigate to the **Teams** tab. Right-click your channel name and select **Workflows**.

2. Find the **Post to a channel when a webhook request is received** workflow and click on it.

3. Give the workflow a name and click **Next**.

4. Select the team and channel, then click **Add workflow**.

5. Copy the **workflow URL** — you will use it when configuring notifications in ThingsBoard. Click **Done**.
</Steps>

## Use the webhook URL in ThingsBoard

When creating a <DocLink product={props.product === Products.EDGE_PE ? Products.PE : props.product === Products.EDGE ? Products.CE : props.product} path="user-guide/notifications">notification</DocLink> recipient of type **Microsoft Teams**, paste the workflow URL and specify the channel name. ThingsBoard will POST notification messages to this URL, and they will appear in the configured Teams channel.

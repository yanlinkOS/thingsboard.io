<Aside type="note">
  Microsoft has deprecated the legacy Connectors (incoming webhooks). Use the Workflows method described below.
</Aside>

## Create a webhook workflow in Microsoft Teams

<Steps>
1. Open <a href="https://teams.microsoft.com" target="_blank" rel="noopener noreferrer">Microsoft Teams</a> and go to the **Teams** tab. Right-click the channel you want to receive notifications and select **Workflows**.
2. Find and click **Post to a channel when a webhook request is received**.
3. Enter a name for the workflow and click **Next**.
4. Select the team and channel, then click **Add workflow**.
5. Copy the generated webhook URL and click **Done**.
</Steps>

## Configure in ThingsBoard

In the <DocLink product={props.product === Products.EDGE_PE ? Products.PE : props.product === Products.EDGE ? Products.CE : props.product} path="user-guide/notifications" target="_blank" bold={false} target="_blank">Notification Center</DocLink>, create a recipient of type **Microsoft Teams** and paste the webhook URL. ThingsBoard will POST notification messages to this URL when a matching notification rule fires, and they will appear in the configured channel.

## IoT Gateway

## Platform Integrations

Your devices already send data to a cloud backend (AWS IoT, Azure, IBM Watson, an MQTT broker, or an OPC-UA server). Platform Integrations bridge that backend to ThingsBoard via a server-side data pipeline. Incoming data is decoded with an **uplink converter** before being stored as device telemetry.

<ImageGallery images={[{ src: '/src/assets/schemas/platform-integrations.svg', alt: 'Platform Integrations architecture', caption: 'Devices already sending data to a cloud backend (AWS IoT, Azure, MQTT broker) are bridged to ThingsBoard via a server-side Integration with uplink converter.' }]} />

<Aside type="note">
  Platform Integrations are a **ThingsBoard Professional Edition** feature.
</Aside>

**ThingsBoard Edge** brings ThingsBoard capabilities to the edge — running on gateways, industrial PCs, or local servers close to your devices. It enables local data collection and processing, offline operation with automatic cloud synchronization, and low-latency decision making at the edge.

## Key Capabilities

- **Local data processing** — collect, store, and analyze device data locally using rule chains, calculated fields, and alarms
- **Offline operation** — continue operating when the cloud connection is lost; data is synchronized automatically when connectivity is restored
- **Cloud synchronization** — push data, alarms, and configuration changes between Edge and the cloud ThingsBoard instance
- **Local dashboards** — access dashboards and manage devices through the Edge web interface without cloud connectivity

## Editions and Compatibility

ThingsBoard Edge is available in two editions. Each Edge edition works **only** with its corresponding ThingsBoard server edition:

| Edge edition | Compatible server | Description |
|---|---|---|
| **ThingsBoard Edge CE** | ThingsBoard CE | Open-source edge computing with core platform features |
| **ThingsBoard Edge PE** | ThingsBoard PE / Cloud | Includes all PE features: integrations, white-labeling, advanced RBAC, groups, and more |

<Aside type="caution">
ThingsBoard Edge CE cannot connect to a ThingsBoard PE server, and vice versa. Always match the Edge edition to your server edition.
</Aside>

### Step 1 — Architecture design

Tell the AI Agent what you're building. It will ask clarifying questions if needed, or jump straight to a proposed solution if your description is clear enough. The proposal breaks down into four sections: Entity Profiles, IAM (user roles and permissions), Calculated Fields, and Alarm Rules. You can review the full entity hierarchy in a visual schema view, request adjustments in plain language, and move on when you're satisfied.

![AI Solution Creator generating IoT architecture design from text prompt](/images/blog/ai-solution-creator/step-1-architecture-design-.webp)

### Step 2 — Dashboard design

The agent immediately generates an overview of the proposed dashboards — what each one will show, how navigation flows between them, and which user roles they're built for. Review, adjust if needed, and confirm.

![AI Solution Creator building IoT dashboard layout automatically](/images/blog/ai-solution-creator/step-2-dashboard-design-1-1.webp)

### Step 3 — Solution installation

Click "Create Solution" and the agent provisions everything in one pass: entities, roles, permissions, emulators, dashboards — all wired together. When it's done, a summary window opens with your solution ready to explore.

![AI Solution Creator deploying the generated IoT prototype](/images/blog/ai-solution-creator/step-3-solution-installation-.webp)

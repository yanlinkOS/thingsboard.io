## What is IoT Hub?

IoT Hub is a marketplace built into the ThingsBoard ecosystem. It provides a growing catalog of ready-to-install items — devices, solution templates, widgets, rule chains, calculated fields, and alarm rules — that can be deployed to your ThingsBoard instance in a single click.

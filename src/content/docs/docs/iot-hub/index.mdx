## What is IoT Hub?

ThingsBoard IoT Hub is a marketplace built into the ThingsBoard ecosystem. It offers a growing catalog of ready-to-install items — devices, solution templates, widgets, rule chains, calculated fields, and alarm rules — that deploy to your ThingsBoard instance in a single click.

IoT Hub is available for **ThingsBoard Community Edition**, **ThingsBoard Professional Edition**, and **ThingsBoard Cloud**.

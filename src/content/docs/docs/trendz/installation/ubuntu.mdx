This guide explains how to install **Trendz Analytics** on **Ubuntu 22.04 LTS**, **Ubuntu 24.04 LTS**, or **Ubuntu 26.04 LTS**.

<Aside type="caution" title="Docker is recommended">
Certain Trendz features — **predictions, metric explorations, and Python calculation fields** — require the Trendz Python Executor, which runs only via Docker. Even on Ubuntu, Docker is needed to fully utilize all Trendz capabilities.

Consider using the <DocLink product={Products.TRENDZ} path="installation/docker">Docker (Linux, macOS)</DocLink> installation guide instead. Proceed here only if you have a specific reason to use an Ubuntu package setup.
</Aside>

## Installation steps

### Step 1. Install Java 17 (OpenJDK)

```bash
sudo apt update && sudo apt install openjdk-17-jdk-headless
```

Set Java 17 as the default version:

```bash
sudo update-alternatives --config java
```

Verify the installation:

```bash
java -version
```

Expected output:

```text
openjdk version "17.x.xx"
OpenJDK Runtime Environment (...)
OpenJDK 64-Bit Server VM (...)
```

### Step 3. Configure database

Trendz uses PostgreSQL. You can install it on the same server or use a managed PostgreSQL service.

**Install PostgreSQL 16:**

```bash
sudo apt install -y postgresql-common
sudo /usr/share/postgresql-common/pgdg/apt.postgresql.org.sh
sudo apt update
sudo apt -y install postgresql-16
sudo service postgresql start
```

Set a password for the `postgres` user:

```bash
sudo -u postgres psql -c "\password"
```

Enter and confirm the password when prompted.

**Create the Trendz database:**

```bash
psql -U postgres -d postgres -h 127.0.0.1 -W
```

```sql
CREATE DATABASE trendz;
```

Press `Ctrl+D` twice to exit.

**Configure the database connection:**

```bash
sudo nano /etc/trendz/conf/trendz.conf
```

Add the following lines. Replace `PUT_YOUR_POSTGRESQL_PASSWORD_HERE` with your actual password:

```bash
# DB Configuration
export SPRING_DATASOURCE_URL=jdbc:postgresql://localhost:5432/trendz
export SPRING_DATASOURCE_USERNAME=postgres
export SPRING_DATASOURCE_PASSWORD=PUT_YOUR_POSTGRESQL_PASSWORD_HERE
```

### Step 4. Run the installation script

```bash
sudo /usr/share/trendz/bin/install/install.sh
```

### Step 5. Start Trendz service

```bash
sudo service trendz start
```

## Troubleshooting

Trendz logs are stored in `/var/log/trendz`. Check for errors with:

```bash
cat /var/log/trendz/trendz.log | grep ERROR
```
